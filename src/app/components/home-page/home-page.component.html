Welcome to the Angular Project-"Personal Contact Diary"

About the Project

This project is a web application built using Angular that serves as a dynamic form for collecting and managing personal information. Users can enter their name, phone number, and email address, which are then displayed inside the table. Additionally, the application allows users to edit and delete these personal details, providing a complete user experience.

Challenges and Difficulties Faced

Throughout the development of this project, we encountered several challenges and difficulties. These included:

  • Handling and validating form data, ensuring data integrity and user-friendliness.
  • Designing a visually appealing and responsive user interface to provide an excellent user experience.
  • Implementing robust functionality for editing and deleting user data cards.
  • Optimizing the performance of the application for a seamless user experience.

Project Explanation

This project showcases the capabilities of Angular for building dynamic and interactive web applications. Here's a brief explanation:

  • Input Form: Users can enter their name, phone number, and email address into the input form.
  • Data Display: The entered data is dynamically displayed as cards within the application.
  • Edit and Delete: Users can edit or delete their data cards to make changes or remove information as needed.
  • Default Values: Users have the ability to view and access a set of predefined values that have been pre-assigned to the table.
  • User Experience: Our project aims to provide a smooth and user-friendly experience for collecting and managing personal information.
